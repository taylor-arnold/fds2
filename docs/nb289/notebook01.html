### Questions

Welcome to the first notebook. The goal today is just to get comfortable running
code, so every question is short. Each gray box is a code cell: click into it and
press `Shift+Enter` to run it. The result shows up right below. To see the result
of something, type it into a cell and run it.

We will start by using Python as a calculator, then create a few variables, and
finish with our first function.

1. Add the numbers 8 and 15.

```{python}
8 + 15
```

2. There are 24 hours in a day. Multiply to find how many hours are in a week of
7 days.

```{python}
24 * 7
```

3. Python follows the usual order of operations, and parentheses let you control
it. Run the expression below and check that the answer is what you expected.

```{python}
(20 - 5) / 3
```

4. Instead of just looking at a result, we can store it. Create a variable named
`mynum` that holds the value of `3 + 5`.

```{python}
mynum = 3 + 5
```

5. Nothing appeared below the cell in the previous question. Why not?

**Answer**: The result was saved into the variable `mynum` instead of being
displayed. Storing a value and showing a value are two different things: an
assignment with `=` keeps the result for later but does not print it.

6. Now use your variable. Add 1 to `mynum`.

```{python}
mynum + 1
```

7. You can have as many variables as you like and combine them. Create a variable
`width` equal to 4 and a variable `height` equal to 7, then multiply them together
to get the area.

```{python}
width = 4
height = 7
width * height
```

8. Time for our first function. The `round` function takes a number and returns
the nearest whole number. Round the value `2.71828`.

```{python}
round(2.71828)
```

9. Most functions accept more than one input. Give `round` a second number to say
how many digits to keep. Round `2.71828` to 2 digits.

```{python}
round(2.71828, 2)
```

10. We can also pass inputs by name, which makes the code easier to read. Round
`2.71828` to 3 digits using the argument names `number` and `ndigits`.

```{python}
round(number=2.71828, ndigits=3)
```
